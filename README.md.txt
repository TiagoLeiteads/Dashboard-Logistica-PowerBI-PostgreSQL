# Dashboard de Logística e Transportes

Projeto de Business Intelligence desenvolvido utilizando PostgreSQL como fonte de dados e Power BI para análise e visualização.

## Objetivo

Demonstrar a construção de uma solução de análise logística utilizando banco de dados relacional e dashboard interativo.

## Arquitetura

PostgreSQL → SQL → Power BI → Dashboard

## Tecnologias Utilizadas

- PostgreSQL
- SQL
- Power BI
- Power Query

## Fonte dos Dados

Base de dados simulada criada para representar operações logísticas e de transporte.
